Subject Plan (High Yield Only)

General Aptitude ≈ 10–12 marks · Daily
Very Easy Quant + Verbal Daily 1 hr
Quant: Percentage, Ratio, Time & Work, Speed–Distance, Averages, simple P&C. Verbal: Synonyms/Antonyms, sentence correction, fill in the blanks, 1 short RC daily.
Engineering Mathematics ≈ 7–8 marks · Daily
Core Linear Algebra Probability
Focus on matrices, rank, determinant, eigenvalues (2×2), system of equations, probability, conditional probability, Bayes, mean and variance, basic calculus.
Digital Logic ≈ 5–6 marks · Days 1–3
Easiest Tech K-maps Combinational
Gates, Boolean algebra, truth tables, K-map (2/3/4 variable), adders, multiplexers, decoders. Aim to finish and then only revise and solve PYQs.
DBMS ≈ 5–6 marks · Days 4–7
High Yield SQL Normalization Transactions
Keys, FDs, 1NF→BCNF, SQL (SELECT, JOIN, GROUP BY, HAVING, aggregates), ACID, conflict serializability. Target: 25+ SQL questions and 10+ normalization problems.
Theory of Computation ≈ 5–6 marks · Days 8–11
Core FA / RE CFG
DFA, NFA, regex→FA, DFA minimization, CFG derivations, ambiguity, parse trees. Skip Turing machine and undecidability in this 30‑day plan.
Compiler Design ≈ 4–5 marks · Days 12–14
Formula Based First/Follow Parsing
Lexical analysis basics, tokens, First & Follow sets, LL(1) idea, LR basics. Focus mostly on First/Follow and simple parsing questions.
Operating Systems ≈ 4–5 marks · Days 15–17
Core Scheduling Deadlock Paging
FCFS, SJF, RR, priority scheduling (waiting/turnaround time), deadlock conditions, Banker's algorithm, paging vs segmentation, FIFO/LRU page replacement.
Computer Networks ≈ 3–4 marks · Days 18–20
Conceptual OSI vs TCP/IP Subnetting
OSI vs TCP/IP, TCP vs UDP, IP addressing & subnetting, CRC basics, Hamming code, a bit of DNS/DHCP. Focus on 15+ subnetting and 10+ error detection problems.
